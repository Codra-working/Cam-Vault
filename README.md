# Cam-Vault
CCTV Web Application

## 소개
이 프로젝트는 CCTV 웹사이트로, 일반 유저가 실시간 영상 확인, 녹화 영상 재생, 이벤트 로그 열람, 사용자 설정을 할 수 있도록 설계되었습니다.

## 기능
- 대시보드: 최근 이벤트 및 카메라 상태 확인
- 실시간 모니터링: 카메라 목록 및 개별 영상 확인
- 녹화 영상: 날짜/시간별 영상 검색 및 재생
- 이벤트 로그: 이벤트 내역 및 상세 보기
- 사용자 설정: 계정 정보 및 알림 설정

## Site Map
```text
CCTV Website
│
├── /login
│    # 로그인 페이지: 아이디/비밀번호 입력, 인증 처리
│
├── /dashboard
│    # 대시보드: 최근 이벤트 요약, 카메라 상태 표시
│
├── /live
│    # 전체 카메라 목록 및 실시간 스트리밍
│    └── /live/camera/{cameraId}
│         # 특정 카메라 영상 확인
│
├── /playback
│    # 날짜/시간별 영상 검색 및 재생
│    └── /playback/camera/{cameraId}?date=YYYY-MM-DD
│         # 특정 카메라의 특정 날짜 영상 조회
│
├── /events
│    # 이벤트 로그 목록 확인
│    └── /events/{eventId}
│         # 특정 이벤트 상세 보기
│
└── /profile
     ├── /profile
     │    # 사용자 프로필 페이지
     └── /profile/settings
          # 계정 정보 및 알림 설정 관리

* 테스트 환경에서는 로컬 스토리지를 스토리지 서버 역할로 설정하여 개발을 진행했습니다.
